Пошаговая настройка в Админке Django
Если ты купил новый вид автоматов и хочешь, чтобы они считались сами:
Шаг 1: Создаем Работу (если её еще нет)
Зайди в Элементы справочника -> Добавить.
Название: "Монтаж 1-полюсного автомата".
Тип: Работа.
Цена: 5$ (твоя цена за установку).
Сохрани.
Шаг 2: Создаем Материал
Название: "Автоматический выключатель 1P" (номинал НЕ ПИШИ, он подставится сам!).
Тип: Материал.
Цена: 4$ (закупочная цена).
Mapping Key: shield_circuit_breaker_1P (строго по шаблону!).
Связанная работа: Выбери из списка работу "Монтаж 1-полюсного автомата", которую создал в Шаге 1.
Сохрани.
4. Почему это удобно (Объясняю «Почему»)
Названия: Ты можешь написать в каталоге название "Авт.выкл. 1P (Китай)". В смету прилетит: "Авт.выкл. 1P (Китай) 16A". Система просто добавит номинал в конец.
Смена цен: Если завтра работа по монтажу подорожает, ты меняешь цену в одном месте (в справочнике работ), и во всех новых импортах цена обновится автоматически.
Связь: Один материал может тянуть за собой одну работу. Этого достаточно для 95% случаев в щитах.
